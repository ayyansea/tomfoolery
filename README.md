# tomfoolery

A useless library with various trinkets, knickknacks and doodahs.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     tomfoolery:
       github: ayyansea/tomfoolery
   ```

2. Run `shards install`

## Usage

```crystal
require "tomfoolery"
```
